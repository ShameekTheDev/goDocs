About

heyDocs is a very simple open source Documentation website. It is already designed and is available in three color themes. You can also customize the theme. The project size is very less and is efficient.

    Customize Theme

    To change the theme follow the instructions given below -
    Open the "style.css" file on your IDE. In the ":root" section, you will see these four root colors -

  • themecolor (this is the primary color)
  • themecolorfaded (this is faded primary color)
  • themepluswhite (this is mixed color made up of primary color + white)
  • headerbgcolor (this is header background color)

    • Edit these root colors according to you and you will get a awesome customised theme.

    Info - If you are using white theme change text colors to black.
